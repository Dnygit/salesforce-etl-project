# Salesforce ETL Data Pipeline Project

## Project Overview
This project simulates Salesforce CRM opportunity data and builds a simple ETL pipeline.

The data is extracted from a CSV file, transformed using Python, and loaded into a MySQL database for analysis.

## Tools Used
- Saleforce CRM
- Python
- MySQL
- SQL
- Git

## Dataset
The dataset contains 100 records of Salesforce-style opportunity data including:

- opportunity_id
- account_name
- stage
- owner
- city
- product
- amount
- close_date


## ETL Workflow
CSV Dataset  
↓  
Python ETL Script  
↓  
MySQL Database  
↓  
SQL Analysis  


## SQL Analysis
Some example queries performed:

- Revenue by product
- Sales by city
- Opportunity pipeline by stage
- Top revenue generating accounts
